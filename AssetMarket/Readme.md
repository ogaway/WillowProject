資産市場実験
=====

資産の価格バブル
-----
投資家は本来の資産価格が反映された価格で資産の売買を行われているのだろうか？  
今回は資産がもたらす期待収益が明らかな場合に被験者がどのように資産の取引を行うかについて分析する。
assetmarket.pyでは参考文献にある実験を再現する。
